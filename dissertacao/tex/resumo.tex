\begin{resumo}
\lettrine[lines=3]{U}\ ma forma de se trabalhar usando \textit{Model Driven Development} - MDD em equipe é usando uma IDE associada à um sistema de versionamento, entretanto, este trabalho propõe o uso de uma wiki para desenvolvimento, de modo que o desenvolvedor é capaz de criar modelos, gerar código-fonte e ainda compartilhar e versionar os modelos, só que de maneira mais simples e fácil, possibilitando que mais usuários não desenvolvedores possam participar do processo de desenvolvimento. Ainda, apenas um navegador é necessário para esta abordagem proposta. 

Para tentar evidenciar de que é possível uma wiki ser usada para desenvolver software, foi criada uma \textit{Domain Specific Language} - DSL em uma wiki e foram realizados 3 estudos de caso que mostram que é viável e que os usuários aceitam desenvolver usando uma wiki.

\end{resumo}
